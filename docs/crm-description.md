# Интеллектуальная CRM-система для ландшафтного бизнеса

## Краткое описание

Полнофункциональная CRM-система нового поколения для компаний в сфере ландшафтного дизайна и благоустройства, построенная на базе искусственного интеллекта. Система автоматизирует все бизнес-процессы: от первого контакта с клиентом до завершения проекта и аналитики.

---

## 📊 Статус реализации функционала

**Обозначения:**
- ✅ **Реализовано** — функция полностью работает в текущей версии
- ⚠️ **Частично реализовано** — базовый функционал работает, требуется доработка
- 📋 **В разработке/планируется** — функция в дорожной карте развития

### 🤖 **11 специализированных AI-помощников**

## Архитектура системы

### Технологический стек

**Frontend:**
- React 18 с TypeScript
- Tailwind CSS для адаптивного дизайна
- Shadcn/ui компоненты
- React Router для навигации
- React Query для управления данными

**Backend:**
- PHP + MySQL на Beget
- Универсальный JSON-слой `crm_records` для CRM-сущностей
- JWT-авторизация в PHP backend
- BegetQueryBuilder на frontend для совместимости со старым Supabase-синтаксисом

**AI интеграции:**
- OpenAI GPT-4o для AI-помощников на PHP backend
- OpenAI TTS для озвучки
- Whisper для распознавания речи

**Внешние интеграции:**
- WhatsApp Business API
- Telegram Bot API
- Мобильные приложения (Capacitor)

#### **Offline режим голосового помощника**
**Описание:** Возможность надиктовывать команды голосовому помощнику без интернета с отложенной обработкой

**Функционал:**
- Локальная запись голосовых команд
- Очередь запросов с сохранением контекста
- Автоматическая обработка при восстановлении связи
- Умное определение качества соединения (не простой navigator.onLine)
- Heartbeat мониторинг с дебаунсом статуса
- Уведомления о режиме работы (online/offline/нестабильно)

**Технические особенности:**
- Heartbeat пинг-запросы с таймаутом 3-5 сек
- Отслеживание успешности последних N запросов
- Дебаунс 10-15 сек перед переключением в offline
- Индикатор качества: хорошая связь / плохая связь / нет связи
- Гистерезис логика (разные пороги для перехода туда и обратно)

**Применение:**
- Диктовка задач без интернета
- Запись заметок по клиентам на объекте
